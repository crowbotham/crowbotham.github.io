About Us

In this class we Generally work on computers and try to learn more about them or we try to improve them as a whole, an example of this is when we used our Raspberry Pi's as mini servers. Our whole goal in this class is to learn more about computers so we can be more adept with them in the future.

Raspberry Pi's are very small computers that are oddly powerful, but are nearly the size of a large phone and generally use a form of Linux, however, it can run a very barebones version of Windows 10 as well! Click here to learn more about it!
